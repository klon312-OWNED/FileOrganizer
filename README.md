# Файловый агент + менеджер файлов

Агент, который сам раскладывает файлы по папкам **по типу и дате загрузки**,
и удобный менеджер, в котором можно быстро найти и открыть нужный файл,
фильтруя по категории и дате.

## Что он делает

- **Сортирует файлы и папки** из «Загрузок», «Рабочего стола» (и любых других
  папок) в папку **«Архив»**, которую создаёт в выбранном тобой месте:

  ```
  <выбранное место>/Архив/
    Картинки/2026/06-Июнь/...
    Документы/2026/05-Май/...
    Видео/2025/12-Декабрь/...
    Папки/2026/06-Июнь/МояПапка/...   (папки переносятся целиком)
    ...
  ```

- **Категории по типу**: Картинки, Видео, Музыка, Документы, Архивы, Программы,
  Код, Папки, Другое (можно настроить).
- **Папки тоже сортируются**: целая папка переносится в категорию «Папки»
  по дате, со всем содержимым внутри (это можно отключить в настройках).
- **Менеджер файлов** с фильтром по категории, году и месяцу, поиском по имени
  и открытием файла или его папки двойным кликом.
- **Два режима работы**:
  - вручную — кнопка «Сортировать сейчас»;
  - фоновый — следит за папками и раскладывает файлы сразу после скачивания
    (работает на событиях, почти не нагружает компьютер).
- **Значок в трее** у фонового агента: правый клик → открыть менеджер,
  сортировать сейчас, включить/выключить слежение, выход.
- **Отмена последней сортировки**: кнопка «Отменить последнюю» вернёт файлы
  и папки на прежние места.
- **Журнал истории**: вкладка «История» показывает, что, откуда и куда было
  перенесено или скопировано; **точки восстановления** — двойной клик или кнопка
  откатывают целую операцию.
- **Несколько режимов сортировки** (в настройках):
  - по типу и дате (по умолчанию);
  - только по типу;
  - только по дате;
  - по расширению;
  - без подпапок (всё в корень архива);
  - **по моим папкам** — умная раскладка в вашу библиотеку категорий.
- **Режим хранения**: перемещать или копировать (оригинал остаётся на месте).
- **Предпросмотр картинок** прямо в окне менеджера при выборе файла.
- **Вкладка «Рабочий стол»** (v1.2): визуальная сетка файлов и папок из
  отслеживаемых папок (Загрузки, Рабочий стол и др.). Отметьте нужные элементы
  и нажмите **«Сортировать выбранное»**, либо **«Сортировать всё»** для полной
  сортировки. Есть **«Выбрать всё»** / **«Снять выбор»** / **«Инвертировать»**,
  **Ctrl+A**, **Ctrl+I** и кнопка **«Обновить»**. С v1.8 элементы **сгруппированы по отслеживаемой папке**.
- **Предпросмотр PDF** (v1.8): первая страница как миниатюра и извлекаемый
  текст (нужен PyMuPDF; без него — подсказка вместо текста).
- **Кэш миниатюр** на диске: `~/.file_organizer/thumbs` — быстрее повторный
  просмотр фото, видео и PDF.
- **Оценка размера ZIP** перед сортировкой, если включено сжатие.
- **Архив**: сортировка по клику на заголовок столбца; контекстное меню
  **«Распаковать ZIP»** для выбранных архивов.
- **Таблицы в Word (.docx)** отображаются в предпросмотре как сетка ячеек.
- **Dry run (тестовый режим)** в настройках: показывает результат сортировки
  без изменения файлов на диске.
- **Подтверждение сортировки с деталями**: количество элементов, папка назначения,
  режим хранения и сжатия.
- **Прогресс долгих операций**: сортировка, сжатие и распаковка показывают
  ход выполнения в строке состояния.
- **Отмена сортировки выбранного** на вкладке «Рабочий стол».
- **Повышена стабильность**: безопасная распаковка ZIP (защита от path traversal),
  защита от пересечения фоновой и плановой сортировки, debounce UI-обновлений
  при частых событиях наблюдателя.

Примеры запросов:

```

найди большие видео

положи docx в Документы/Учёба/Python
сжми установщики в zip
все pdf за 2025 год отсортируй
сортируй по типу
```

Как включить LLM: вкладка **ИИ-помощник** → **Настройки ИИ** → OpenAI API или Ollama
(по умолчанию работают локальные правила без сети).

### Новое в v1.13.0 программа изучает ваши подпапки
  (имена, расширения, примеры файлов, текст из .txt/.md/.docx/.pdf) и
  предлагает, куда положить новые файлы с рабочего стола или загрузок.
- Кнопка **«По моим папкам»** на вкладке «Рабочий стол» и режим
  **«По моим папкам»** в настройках сортировки.
- Диалог подтверждения: файл → папка (с оценкой), правка вручную,
  создание новой папки, «Другое» или пропуск.
- Настройки: корень библиотеки, порог совпадения, авто-предложение папок.

#### Как пользоваться «Мои папки»

1. Создайте папку-библиотеку, например `Документы/Учёба`, с подпапками
   по темам: `Курс Python`, `Математика`, `Курсовая 2025`.
2. Положите в подпапки несколько типичных файлов — программа построит
   профиль каждой категории.
3. В **Настройки сортировки** → **Умная раскладка по моим папкам** укажите
   корень библиотеки и при необходимости порог (по умолчанию 0.28).
4. На вкладке **Рабочий стол** выберите файлы и нажмите **По моим папкам**.
5. Проверьте предложенное сопоставление в диалоге и нажмите
   **Применить раскладку**.

Ограничения: сопоставление эвристическое (без облачного ИИ); для .doc
  без текста учитываются только имя и расширение; папки целиком не
  раскладываются — только файлы; перед перемещением всегда нужно
  подтверждение; при совпадении имён создаётся `файл (1).ext`.

### ИИ-помощник

Вкладка **«ИИ-помощник»** анализирует отслеживаемые папки и архив и предлагает:

- что сортировать сейчас (кандидаты на раскладку);
- оптимальный режим `sort_mode` для вашего набора файлов;
- включение сжатия для крупных установщиков;
- кандидатов на умную уборку (эвристика из `classify.py`).

В строке ввода можно писать запросы по-русски — и поиск, и сортировку.
Сначала показывается **план** (dry-run), затем кнопка **Применить план**.
Результаты и советы — с кнопками **Сорт.**, **Откр.**, **Искл.**, **Папк.**

**Настройка API / Ollama** (кнопка «Настройки ИИ» на вкладке):

| Режим | Что указать |
|-------|-------------|
| Локальные правила | ничего — работает сразу |
| OpenAI API | API-ключ, Base URL (напр. `https://api.openai.com/v1`), модель |
| Ollama | URL (по умолчанию `http://localhost:11434`), имя модели |

⚠ Ключ хранится в открытом виде в `settings.json` — не публикуйте этот файл.

### Вкладка «Поиск по всему ПК»

Отдельная вкладка ищет файлы выбранной категории **по всем папкам компьютера**
(например, все документы Word), а не только в архиве:

- выбери категорию (или задай расширения, напр. `.docx,.doc`) и нажми
  «Искать по ПК»;
- результаты появляются по мере поиска, есть фильтр по имени и кнопка «Стоп»;
- при выборе файла справа показывается **предпросмотр содержимого**: текст,
  Word (.docx), Excel (.xlsx), картинки;
- кнопки действий: **Открыть**, **Открыть папку с файлом**, **Копировать в...**,
  **Переместить в...**, **Удалить** (в корзину);
- **множественный выбор**: Ctrl+клик / Shift+клик — копировать, переместить или
  удалить сразу несколько файлов;
- **подсветка кандидатов на удаление**: красным выделяются файлы, которые,
  скорее всего, можно удалить (личное/мусор: установщики, временные файлы,
  дубликаты, скриншоты, старые медиа в загрузках). Учебные/рабочие документы
  (Word, Excel, PDF, по ключевым словам и рабочим папкам) защищены и не
  помечаются. Кнопка **«Выделить кандидатов»** выделяет их все разом, галочка
  **«Только кандидаты на удаление»** показывает лишь их. Решение всегда за тобой.

Системные и служебные папки (Windows, Program Files, AppData, node_modules
и т.п.) при поиске пропускаются для скорости.

## Установка

### Вариант 1: готовый установщик (рекомендуется)

1. Скачайте **FileOrganizer-Setup-…-win64.zip** с [страницы релизов](https://github.com/klon312-OWNED/FileOrganizer/releases).
2. Распакуйте архив.
3. Запустите **`install.bat`** — программа установится в
   `%LOCALAPPDATA%\Programs\FileOrganizer`, на рабочий стол появится ярлык.
4. Удаление: запустите **`Удалить.bat`** из папки установки или
   `installer\uninstall.ps1`.

Python устанавливать **не нужно** — в сборку входят все зависимости.

### Вариант 2: из исходников (для разработки)

Нужен Python 3.10+. Установите зависимости:

```powershell
python -m pip install -r requirements.txt
```

Сборка exe и ZIP-установщика:

```powershell
powershell -ExecutionPolicy Bypass -File build\make_release.ps1
```


## Запуск

- **Менеджер файлов (окно):** двойной клик по `Запустить менеджер.bat`
  или `python run.py`.
- **Фоновый агент (без окна):** двойной клик по `Фоновый агент.bat`.
- **Автозапуск при включении ПК:** `Добавить в автозагрузку.bat`
  (отключить — `Убрать из автозагрузки.bat`).

## Как пользоваться менеджером

1. Нажмите **«Настройки сортировки»**: выберите **схему раскладки**, режим
   **перемещения или копирования**, дату для папок, место архива и отслеживаемые
   папки.
2. Нажмите **«Сортировать сейчас»** — файлы разложатся по выбранной схеме.
3. Вкладка **«Рабочий стол»** — визуально просмотрите файлы в отслеживаемых
   папках, отметьте нужные и нажмите **«Сортировать выбранное»** (или
   **«Сортировать всё»**).
4. Вкладка **«История»** — журнал всех операций: откуда → куда, с отменой любой
   пакетной сортировки.
5. Слева выберите **категорию / год / месяц**, сверху можно **искать по имени**.
6. **Двойной клик** открывает файл. Правый клик → «Показать в папке».

## Где хранятся данные

- Настройки: `~/.file_organizer/settings.json`
- Индекс файлов: `~/.file_organizer/index.db`
- Кэш миниатюр: `~/.file_organizer/thumbs`

## Примечания

- Файлы, которые ещё качаются (`.crdownload`, `.part`, `.tmp` и т.п.),
  не трогаются, пока загрузка не завершится.
- Если файл с таким именем уже есть в папке назначения, к новому добавляется
  ` (1)`, ` (2)` и т.д. — ничего не перезаписывается.
- «Дата загрузки» берётся из времени появления файла (на Windows — время
  создания файла).
